Mis Aprendizajes en Computación

En el área de computación ya tenía algunos conocimientos básicos, pero durante este primer mes pude reforzar y profundizar en varios temas importantes en el curso de Algoritmos y Programación Básica.

1. Diagramas de Flujo

Los diagramas de flujo fueron mi primera introducción a la lógica de programación. Aprendí a representar algoritmos de manera visual antes de escribir código.

Reflexión

Entendí que antes de programar, es crucial planificar. Los diagramas de flujo ayudan a visualizar el problema y su solución antes de escribir una sola línea de código.

2. Loops (Ciclos) en Python

Los bucles fueron un concepto revelador. Descubrí cómo automatizar tareas repetitivas con pocas líneas de código.

Loop For - Ejemplo del curso

# Ejercicio: Imprimir números del 1 al 5 for i in range(1, 6): print(f"Número: {i}") # Ejemplo: Recorrer una lista frutas = ["manzana", "banano", "uva"] for fruta in frutas: print(f"Me gusta la {fruta}")

Loop While

# Contar hacia atrás contador = 5 while contador > 0: print(contador) contador -= 1 print("¡Despegue!")

Ciclos Anidados - Ejemplo con Turtle

# Dibujo con ciclos anidados import turtle for i in range(6): for j in range(6): turtle.forward(50) turtle.right(60) turtle.right(60)

Reflexión

Los loops me enseñaron el poder de la automatización. Imaginar cuántas líneas de código necesitaría si tuviera que escribir cada iteración manualmente... ¡los bucles hacen posible lo imposible!

3. Chatbot UVG

Durante este primer mes también trabajé en un proyecto personal: un chatbot para la Universidad del Valle de Guatemala. Este proyecto me permitió aplicar mis conocimientos de programación web.

🌐 Tecnología

HTML, CSS (Tailwind), JavaScript y integración con OpenRouter API para conectar con modelos de IA como Claude y GPT.

✨ Características

Interfaz de chat moderna, soporte para múltiples modelos de IA, modo oscuro, y personalización de la personalidad del asistente.

🎯 Objetivo

Crear un asistente virtual para ayudar a estudiantes de la UVG con dudas académicas y información sobre la universidad.

📚 Aprendizajes

Trabajo con APIs externas, manejo de datos JSON, diseño de interfaces de usuario y pensamiento lógico.
