🚀 Inicio Rápido

1 Obtén tu API Key de OpenRouter

Visita openrouter.ai/keys y crea una cuenta gratuita. Copia tu API key (comienza con sk-or-v1-)

2 Abre el Chatbot

Abre el archivo index.html en tu navegador web favorito

3 Configura tu API Key

Pega tu API key en el panel de configuración que aparece y haz clic en "Guardar"

4 ¡Comienza a Conversar!

Escribe tu primer mensaje y presiona Enter o haz clic en "Enviar"

⚙️ Configuración Detallada

Obtener API Key de OpenRouter

OpenRouter es un servicio que te permite acceder a múltiples modelos de IA con una sola API key:

  1. Ve a openrouter.ai
  2. Crea una cuenta con tu email
  3. Ve a la sección "Keys" en tu dashboard
  4. Haz clic en "Create Key"
  5. Copia la key generada (guárdala en un lugar seguro)
  6. Agrega créditos a tu cuenta (mínimo $5 USD)
💡 Tip: OpenRouter ofrece $1 USD gratis para probar el servicio. Perfecto para empezar.

💬 Cómo Usar el Chatbot

Escribir Mensajes

  • Escribe tu mensaje en el campo de texto inferior
  • Presiona Enter para enviar
  • O haz clic en el botón "Enviar"

Funcionalidades

  • Historial de Conversación: El chatbot recuerda toda la conversación actual
  • Indicador de Escritura: Verás tres puntos animados mientras el asistente piensa
  • Scroll Automático: La conversación se desplaza automáticamente a los nuevos mensajes
  • Persistencia de API Key: Tu API key se guarda localmente (no se comparte)

🎨 Personalización

Cambiar el Modelo de IA

Edita el archivo index.html y busca la línea:

model: 'anthropic/claude-3.5-sonnet',

Cámbiala por:

model: 'openai/gpt-3.5-turbo', // Más económico // o model: 'google/gemini-pro', // Balance precio/calidad

Modificar la Personalidad

Busca la variable SYSTEM_PROMPT en index.html y edita el texto para cambiar:

  • Tono de conversación (formal/informal)
  • Conocimientos específicos
  • Estilo de respuesta
  • Uso de emojis

Cambiar Colores

Modifica las variables CSS en la sección <style>:

/* Colores principales */ background: linear-gradient(135deg, #1e3a8a 0%, #3b82f6 100%); /* Color de mensajes del usuario */ .message.user .message-content { background: linear-gradient(135deg, #3b82f6 0%, #2563eb 100%); }

🔧 Solución de Problemas

Error: "Invalid API Key"

⚠️ Solución:
  • Verifica que tu API key comience con sk-or-v1-
  • Asegúrate de haber copiado la key completa
  • Verifica que tengas créditos en tu cuenta de OpenRouter

El chatbot no responde

  • Abre la consola del navegador (F12) y busca errores
  • Verifica tu conexión a internet
  • Intenta recargar la página
  • Borra el localStorage y vuelve a configurar la API key

Respuestas muy lentas

  • Considera cambiar a un modelo más rápido (GPT-3.5 Turbo)
  • Reduce el parámetro max_tokens en el código
  • Verifica tu conexión a internet

Costos muy altos

  • Cambia a un modelo más económico (GPT-3.5 o Gemini Pro)
  • Reduce max_tokens para respuestas más cortas
  • Limpia el historial de conversación periódicamente
💡 Tip de Ahorro: Una conversación típica cuesta entre $0.01 y $0.05 USD. Con $5 USD puedes tener ~100-500 conversaciones dependiendo del modelo.
